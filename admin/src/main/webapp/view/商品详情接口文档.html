商品分类列表接口

接口地址：

http://test.api.open.huobanplus.com/goods/findByGoodsIdAndUnionId

支持格式：

json

请求方式：

get

请求参数：

  名称 类型 必须 说明
  goodsId Integer   是 商户主键id
  userId Integer   否 会员主键id

接口备注：

返回商品的所有信息

json字段：

  名称 类型 必有 说明
  id Integer  是 商品主键id
  title String  是 商品名称
  sales Integer  是 销量
  marketPrice double  是 市场价
  price double  是 销售价
  vipPrice double  否 会员价
  specification String  是 商品规格
  tags String  是 商品标签（包邮，七天无理由退换货等）
  stock Integer  是 库存
  thumbnail Array  是 商品介绍图片url
  detail String  是 商品详情介绍

个人中心接口

接口地址：

http://test.api.open.huobanplus.com:8081/personal/findDetailByUnionId

响应格式：

json

请求方式：

get

请求参数：

  名称 类型 必须 说明
  merchantId Integer   是 商户主键id
  UnionId Integer   是 用户主键id
  Sex Boolean   是 性别
  nickName String   是 用户昵称
  merchantId Integer   是 商户主键id
  City String   否 所在城市
  Province String   否 所在省份
  Country String   否 所在国家
  Headimgurl String   是 Headimgurl
