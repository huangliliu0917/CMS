签名算法：

    第一步，设所有发送或者接收到的数据为集合M，将集合M内非空参数值的参数
    (不包括sign)按照参数名ASCII码从小到大排序（字典序），使用URL键值对的格式
    （即key1value1key2value2…）拼接成字符串stringA。

    特别注意以下重要规则：
    · 参数名ASCII码从小到大排序（字典序）；
    · 如果参数的值为空不参与签名；
    · 参数名区分大小写；

    第二步，在stringA最后拼接上key（接口密钥）得到stringSignTemp字符串，
    并对stringSignTemp进行MD5运算，得到sign值signValue。

    举例：
        假设传送的参数如下：

        appid：	wxd930ea5d5a258f4f
        mch_id：	10000100
        device_info：	1000
        body：	test

        第一步：对参数按照keyvalue的格式，并按照参数名ASCII字典序排序如下：
        stringA="appidwxd930ea5d5a258f4fbodytestdevice_info1000mch_id10000100";

        第二步：拼接API密钥（假设密钥为123）：
        stringSignTemp="appidwxd930ea5d5a258f4fbodytestdevice_info1000mch_id10000100123"
        sign=MD5(stringSignTemp)="9A0A8659F005D6984697E2CA0A9CF3B7"

    最终发送的参数为：

        appid：	wxd930ea5d5a258f4f
        mch_id：	10000100
        device_info：	1000
        body：	test
        sign：9A0A8659F005D6984697E2CA0A9CF3B7

请求方式：

请求参数：

  customerId Integer   是 商户主键id
  sign String   是 签名

接口备注：

如果该分类还有子分类，则应该同时返回一级子分类的数据

返回示例：

            "picUrl": "http://www.cms.com/1.jpg"
        },
        {
            "goodsTypeId": 3,
            "id": 4,
            "name": "护肤彩妆",
            "picUrl": "http://www.cms.com/4.jpg"
        }
    ],
    "msg": "数据获取成功"
}

json字段：

  名称 类型 必有 说明
  code Integer  是 状态码（请求数据成功应返回200状态码）
  msg String  是 请求状态说明
  data Array  否 数据内容
  id Integer  是 商品分类主键id
  name String  是 商品分类名称
  goodsTypeId Integer  否 商品类型id
  picUrl String  是 完整的图片url
  children Array  否 该商品分类下的所有一级子分类

商品检索接口

接口地址：

待补充

支持格式：

json

请求方式：

get

请求参数：

  名称 类型 必须 说明
  customerId Integer   是 商户主键id
  goodsCatId Integer   否 商品分类主键id
  goodsTypeId Integer   否 商品类型主键id
  brandId Integer   否 品牌主键id
  minPrice Integer   否 起始价格
  maxPrice Integer   否 结束价格
  userId Integer   否 会员主键id（用于查询会员价）
  keyword String   否 商品关键字（可模糊匹配）
  pageNo Integer   否 页码
  sortEnum Integer   否 排序依据（0:销量,1:价格,2；上架时间）
  direction Integer   否 升降序（0:降序,1:升序）
  sign String   是 签名

接口备注：

默认返回第一页，按上架时间降序排序

                "salePrice": 145,
                "sales": 45,
                "shelveTime": "2016-3-22 18:34:33",
                "thumbnail": "http://www.cms.com/11.jpg",
                "vipPrice": 135
            },
            {
                "id": 2,
                "marketPrice": 399,
                "name": "花花公子女士牛仔裤女秋经典简约修身潮流小脚女裤牛仔长裤子",
                "salePrice": 199,
                "sales": 88,
                "shelveTime": "2016-3-20 18:34:33",
                "thumbnail": "http://www.cms.com/12.jpg",
                "vipPrice": 179
            }
        ],
        "totalPages": 2,
        "totalRecords": 3
    },
    "msg": "数据获取成功"
}

json字段：

  名称 类型 必有 说明
  code Integer  是 状态码（请求数据成功应返回200状态码）
  msg String  是 请求状态说明
  data Object  否 数据内容
  currentPageNo Integer  是 当前页码
  totalPages Integer  是 总页数
  totalRecords Integer  否 总记录数
  records Array  否 检索结果集
  id Integer  是 商品主键id
  name String  是 商品名称
  shelveTime String  是 上架时间（日期格式:"yyyy-MM-dd HH:mm:ss"）
  sales Integer  是 销量
  marketPrice double  是 市场价
  salePrice double  是 销售价
  vipPrice double  否 会员价
  thumbnail String  是 缩略图完整url
