商品分类列表接口

接口地址：

http://api.open.huobanplus.com/categories/search/findByMerchantId

支持格式：

json

请求方式：

get

请求参数：

  名称 类型 必须 说明
  merchantId Integer   是 商户主键id

接口备注：

如果该分类还有子分类，则应该同时返回一级子分类的数据

返回示例：

[
    {
        "id": 1,
        "name": "服装内衣",
        "children": [
            {
                "id": 2,
                "name": "女装",
                "goodsTypeId": 2,
                "picUrl": "http://www.cms.com/2.jpg"
            },
            {
                "id": 3,
                "name": "男装",
                "goodsTypeId": 2,
                "picUrl": "http://www.cms.com/3.jpg"
            }
        ],
        "goodsTypeId": 1,
        "picUrl": "http://www.cms.com/1.jpg"
    },
    {
        "id": 4,
        "name": "护肤彩妆",
        "goodsTypeId": 3,
        "picUrl": "http://www.cms.com/4.jpg"
    }
]

json字段：

  名称 类型 必有 说明
  id Integer  是 商品分类主键id
  title String  是 商品分类名称
  goodsTypeId Integer  否 商品类型id
  picUrl String  是 完整的图片url
  children Array  否 该商品分类下的所有一级子分类

商品检索接口

接口地址：

待补充

支持格式：

json

请求方式：

get

请求参数：

  名称 类型 必须 说明
  merchantId Integer   是 商户主键id
  goodsCatId Integer   否 商品分类主键id
  goodsTypeId Integer   否 商品类型主键id
  brandId Integer   否 品牌主键id
  minPrice Integer   否 起始销售价格(闭合，登录后查询依据变为会员价)
  maxPrice Integer   否 结束销售价格(闭合，登录后查询依据变为会员价)
  userId Integer   否 会员主键id（用于查询会员价）
  keyword String   否 商品关键字（可模糊匹配）
  page Integer   否 页码
  sort String   否 排序（propertyName[,desc|asc] 默认asc）

接口备注：

默认返回第一页，按权重升序排序

返回示例：

{
    "elements": [
        {
            "id": 1,
            "title": "KROP女装抓毛长裙卫衣",
            "shelveTime": "2016-3-22 18:34:33",
            "sales": 45,
            "marketPrice": 299,
            "price": 145,
            "vipPrice": 135,
            "thumbnail": "http://www.cms.com/11.jpg"
        },
        {
            "id": 2,
            "title": "花花公子女士牛仔裤女秋经典简约修身潮流小脚女裤牛仔长裤子",
            "shelveTime": "2016-3-20 18:34:33",
            "sales": 88,
            "marketPrice": 399,
            "price": 199,
            "vipPrice": 179,
            "thumbnail": "http://www.cms.com/12.jpg"
        }
    ],
    "page": {
        "number": 0,
        "totalPages": 2,
        "totalElements": 3,
        "size": 2
    }
}

json字段：

  名称 类型 必有 说明
  elements Array  是 检索结果集
  number Integer  是 当前页码(0表示第一页)
  totalPages Integer  是 总页数
  totalElements Integer  是 总记录数
  id Integer  是 商品主键id
  title String  是 商品名称
  shelveTime String  是 上架时间（日期格式:"yyyy-MM-dd HH:mm:ss"）
  sales Integer  是 销量
  marketPrice double  是 市场价
  price double  是 销售价
  vipPrice double  否 会员价
  thumbnail String  是 缩略图完整url
